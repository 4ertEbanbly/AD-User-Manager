AD-User-Manager
==========================================
PowerShell скрипт для автоматического управления пользователями Active Directory с системой логирования.
### Функциональные возможности

Основные операции
* Автоматическое определение домена - скрипт самостоятельно определяет текущий домен и его структуру
* Создание пользователей - интерактивное создание учетных записей с валидацией данных
* Удаление пользователей - безопасное удаление с подтверждением операции
* Сброс паролей - генерация сложных паролей и принудительная смена при следующем входе
* Статистика - просмотр информации о пользователях в выбранных подразделениях

Безопасность и логирование
- Cоздание логов - уникальные файлы при каждом запуске
- SecureString - обработка паролей
- Валидация данных - проверка email, телефонных номеров и уникальности логинов

### Требования
- Windows Server (контроллер домена)
- PowerShell 5.1+
- Модуль Active Directory
- Права на создание/удаление пользователей в домене

Использование
-------------------------
При запуске скрипт автоматически определит домен и предложит выбрать действие:

1. Создать нового пользователя
2. Удалить пользователя  
3. Сбросить пароль пользователя
4. Показать статистику
5. Выход

### Создание пользователя
* Автоматическая проверка уникальности логина
* Выбор подразделения (OU) из доступных в домене
* Назначение групп безопасности
* Генерация сложного пароля или ручной ввод

### Логирование
Все операции записываются в лог-файлы по пути:
C:\Users\[Username]\Documents\ADscript\logs\logs_AD_create_delete_YYYYMMDD_HHMMSS.txt
